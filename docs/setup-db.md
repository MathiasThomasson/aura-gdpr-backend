# PostgreSQL Setup for AURA-GDPR

Run these commands as postgres:

CREATE DATABASE aura;
CREATE ROLE "user" WITH LOGIN PASSWORD 'CHANGEME_PASSWORD';
GRANT ALL PRIVILEGES ON DATABASE aura TO "user";

\c aura

GRANT ALL PRIVILEGES ON ALL TABLES IN SCHEMA public TO "user";
GRANT ALL PRIVILEGES ON ALL SEQUENCES IN SCHEMA public TO "user";
